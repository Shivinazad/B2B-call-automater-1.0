# 🎨 Visual Design Guide - IndiaMART Sniper

## Design Philosophy
- **Dark Theme**: Primary background #0a0a0f (almost black)
- **Glassmorphism**: Translucent cards with backdrop blur
- **Accent Colors**: 
  - Purple: #8b5cf6
  - Cyan: #06b6d4
  - Pink: #ec4899
- **Typography**: System fonts, clean and minimal
- **Animations**: Subtle, smooth, professional

## Color Usage Guide

### Cards & Surfaces
- Main cards: `glass` class (semi-transparent with blur)
- Borders: `border-dark-border` (#1e1e2e)
- Hover states: Brighter borders with glow

### Status Indicators
- Pending: Gray (`text-gray-500`)
- In Progress: Purple glow (`text-accent-purple`, animated)
- Completed: Green (`text-green-500`)
- Error: Red (`text-red-500`)

### Buttons
- Primary: Gradient purple to pink
- Secondary: Glass with border
- Hover: Scale + shadow glow

### Text Hierarchy
- Headings: Gradient text (purple → pink → cyan)
- Body: `text-white`
- Secondary: `text-gray-300`
- Tertiary: `text-gray-400` or `text-gray-500`

## Animation Patterns

### Entrance Animations
```typescript
initial={{ opacity: 0, y: 20 }}
animate={{ opacity: 1, y: 0 }}
transition={{ duration: 0.6 }}
```

### Progress Bars
```typescript
// Growing bar
animate={{ width: '100%' }}
transition={{ duration: 3, ease: 'linear' }}

// Shimmer effect
animate={{ x: ['-100%', '400%'] }}
transition={{ repeat: Infinity }}
```

### Status Pulses
```typescript
className="glow-pulse"
// Defined in globals.css
```

### Button Interactions
```typescript
whileHover={{ scale: 1.02 }}
whileTap={{ scale: 0.98 }}
```

## Typography Scale

- Heading 1: `text-5xl font-bold` (48px)
- Heading 2: `text-4xl font-bold` (36px)
- Heading 3: `text-2xl font-bold` (24px)
- Body Large: `text-lg` (18px)
- Body: `text-base` (16px)
- Small: `text-sm` (14px)
- Tiny: `text-xs` (12px)

## Spacing System

- Section gaps: `space-y-6` or `space-y-8`
- Card padding: `p-6` or `p-8`
- Margins: Consistent use of `mb-4`, `mb-6`, `mb-12`
- Grid gaps: `gap-4` or `gap-6`

## Responsive Breakpoints

- Mobile: Default (< 768px)
- Tablet: `md:` (≥ 768px)
- Desktop: `lg:` (≥ 1024px)
- Wide: `xl:` (≥ 1280px)

Example:
```tsx
className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3"
```

## Icon Usage

- Lucide React icons throughout
- Size: `w-5 h-5` for inline, `w-8 h-8` for featured
- Color: Matches surrounding context or accent

## Key UX Principles

1. **Immediate Feedback**: Every action has visual response
2. **Clear States**: Obvious pending/active/complete states
3. **Breathable Space**: Dark backgrounds, plenty of padding
4. **Smooth Transitions**: 0.3-0.6s duration for most animations
5. **Consistent Patterns**: Same interaction patterns throughout

---

This guide ensures consistent visual design across the entire application.
